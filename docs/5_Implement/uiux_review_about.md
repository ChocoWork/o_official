## ペルソナ適合サマリー

ABOUT は**コアペルソナの「信頼」軸が最も効くページ**。中原玲央は「なぜこの服を作ったか」「製造工程の透明性」「受注生産＝安心」に金を払う。現状は Brand Philosophy / Quality の2節のみで**ブランドの差別化（天然繊維100%・洗える・受注生産・ユニセックス・SS/AW固定）が一切書かれておらず、購入理由が完成しない**。レイアウトの完成度は高いが、コンテンツが薄いのが致命的。

## 良い点（維持）

- `lg:grid-cols-[38.2fr_61.8fr]` で**黄金比分割を正しく実装**（implement-uiux Layer 4 準拠）。
- 画像 alt が具体的（[page.tsx:36](../../src/app/about/page.tsx#L36) 等）でWCATG適合。HOMEのヒーロー alt より良い手本。
- hover scale・duration 500ms の抑制された演出はブランド適合。

---

## implement-uiux に「あえて従わない」判断

- **過度な図解・アイコン装飾を足さない**: スキルは情報の視覚化を推奨するが、ABOUT は「静かで強い世界観」を文章と余白で語るのが本ブランドの作法。A-1 の追加節も**アイコン多用ではなく Didot 見出し + 端正な本文**で構成する。
- それ以外（CTA追加・コントラスト・具体情報）はスキル指針どおり従うべき。

## 修正反映（2026-06-20）

- **A-1**: 「Our Commitments」節を新設し、天然繊維100% / 受注生産・シーズンレス / ユニセックス・フリーサイズ の3本柱を本文化（肌と地球へのやさしさ・自然分解、つくりすぎない・大量廃棄への解、古着になっても次の誰かへ）。
- **A-2**: 末尾に締めのステートメント + CTA（VIEW COLLECTION → /item、LOOKBOOK → /look）を追加。
- **A-3**: h1 直下にタグライン「時を紡ぐニュートラルモードな日常着」を追加。
- **創業者の言葉**（AskUserQuestion で「両方」を選択）: ブランド主語の Philosophy 説明＋一人称の創業者引用「WHY WE MAKE / なぜ、つくるのか」を新設。
- **A-5（対応中）**: 厚みのある生地・洗える等の具体性は追加したが、**個別商品の素材名・産地は PDP の構造化（ID-1）側で対応予定**。
- **A-4（未対応）**: 画像は about.png / mainphoto.png を流用のまま。制作工程・素材アップの実写があれば差し替え推奨。

## 修正反映（2026-06-23）

- **A-4（対応済）**: Philosophy / Quality の各画像を `<figure>` + `<figcaption>` 化し、ブランドの事実（天然繊維100%／受注生産／シーズンレス、厚手の天然素材／丁寧な縫製／家庭で洗える）をキャプション化。「同一画像＝情報量ゼロ（反復の悪用）」の指摘を情報付与で解消。**注: 物理的な写真差し替え（制作工程・素材アップの実写）は資産が用意でき次第の対応を引き続き推奨**（about.png / mainphoto.png は暫定流用）。
- **A-5（対応済）**: Quality & Craftsmanship に事実ベースの一文（天然繊維100%・古着のような厚みとハリ・家庭の洗濯機で洗える）を追加し、抽象形容詞中心の状態を解消（個別商品の素材名・産地は引き続き PDP 側）。
- **A-6（対応済）**: h1 を `clamp(2rem,4vw,3rem)` → サイト共通トークン `var(--lk-size-4xl)` に統一。
- **A-7（対応済）**: Philosophy/Quality 各節にブランドのマイクロ見出し（OUR STORY / MATERIAL & MAKING、Commitments と同じ `about-secondary-title`）を追加し、縦積み時のリズム・スキャン性を改善。
- **A-8（対応済）**: ルートコンテナに段階的な外周余白 `px-1 sm:px-4 lg:px-8`（グローバル `main` の `px-5` に加算）を付与し、広めの `max-w-6xl` でも端の詰まりを回避。
- 実装: [src/app/about/page.tsx](../../src/app/about/page.tsx)。

## 総評（ABOUT）

レイアウト品質は高いが、**ブランドの差別化が言語化されていない（A-1）/締めのCTAが無い（A-2）**ことで、最も重要な「信頼→購入」の橋渡しができていない。コンテンツ拡充が最優先。
