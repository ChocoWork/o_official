# 概要

管理画面でのプロモーション作成、ウィッシュリスト保存（匿名/会員）、入荷通知登録、メール連携を実装する。

# 詳細

- エンドポイント例: `POST /api/promotions`, `POST /api/wishlist`, `POST /api/items/:id/notify`
- ウィッシュリストは匿名は Cookie、会員は DB に保存。入荷通知はメール送信トリガーを含む。

# 受け入れ条件

1. プロモーション作成が管理画面で可能であり割引が正しく計算される。
2. ウィッシュリストが匿名→会員で同期される。
3. 入荷通知登録でメールが送信される（またはキューに登録される）。

# 実装ノート

- メールテンプレートはブランドガイドに従う。

# 依存関係

- SendGrid 等のメールプロバイダ
